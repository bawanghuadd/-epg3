## 视觉设计

### 颜色方案
- **主色调**: 渐变紫色 (#667eea → #764ba2)
- **客户标识**: 绿色 (#4CAF50)
- **工程师标识**: 橙色 (#FF9800)
- **成功提示**: 绿色
- **警告提示**: 橙色
- **背景色**: 浅灰 (#f5f5f5)
- **卡片背景**: 白色 (#ffffff)
- **文本主色**: 深灰 (#333333)
- **文本辅色**: 中灰 (#666666)
- **文本提示**: 浅灰 (#999999)

### 交互反馈
1. **按钮点击**
   - 透明度变化（active: 0.7）
   - 显示加载状态

2. **面板动画**
   - 滑入动画（0.3s ease-out）
   - 淡入动画（0.3s ease-out）

3. **Toast 提示**
   - 登录成功：✓ 图标
   - 操作失败：× 图标
   - 普通提示：无图标

4. **Loading 提示**
   - 遮罩层
   - "登录中..." 文字
   - 旋转动画

## 使用流程

### 流程 1：首次登录
```
打开应用
    ↓
显示登录页面
    ↓
点击"模拟客户登录"
    ↓
显示 Loading
    ↓
模拟延迟 800ms
    ↓
保存 Token 和用户信息
    ↓
显示 Toast "登录成功"
    ↓
延迟 1500ms
    ↓
跳转到客户首页
```

### 流程 2：快速切换用户
```
当前已登录状态
    ↓
点击右下角 👤 按钮
    ↓
弹出用户列表面板
    ↓
选择要切换的账号
    ↓
快速切换（延迟 300ms）
    ↓
显示 Toast "登录成功"
    ↓
刷新页面数据
    ↓
跳转到对应首页
```

### 流程 3：测试页面操作
```
访问测试页面
    ↓
查看当前登录状态
    ↓
点击快速登录卡片
    ↓
切换到指定账号
    ↓
查看日志记录
    ↓
测试工具函数
    ↓
执行测试场景
```

## 响应式适配

### 小屏幕（< 375px）
- 按钮字体缩小
- 头像尺寸缩小
- 间距适当减小

### 中等屏幕（375px - 414px）
- 标准尺寸显示
- 最佳视觉效果

### 大屏幕（> 414px）
- 保持标准尺寸
- 内容居中显示

## 可访问性

1. **清晰的视觉层次**
   - 标题明显
   - 内容分组清晰
   - 重要信息突出

2. **足够的触摸区域**
   - 按钮最小高度 80rpx
   - 列表项最小高度 100rpx
   - 间距充足便于点击

3. **明确的状态反馈**
   - 加载状态显示
   - 操作结果提示
   - 当前选中高亮

4. **错误处理友好**
   - 失败原因明确
   - 提供重试机制
   - 不阻断用户操作
